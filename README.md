# Smiles automáticos

### Este archivo lo que va a hacer:
* Crear una tabla llamada wd_smiles
* Insertar datos dentro de la tabla

No repite, ni vuelve a insertar el mismo contenido, en el caso que en la carpeta haya 10 smiles, entonces en la tabla se insertarán 10 registros

### La tabla cuenta con:
* wd_id = La ID para identificar el smile
* wd_router = La url de la carpeta de smiles
* wd_smile = Nombre de la imagen+extensión
* wd_date = Cuando fue creado
